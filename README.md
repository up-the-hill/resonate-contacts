# Resonate Technical Exercise - Contacts App

Built using Vite + React.

## Steps to run

### npm

```sh
git clone https://github.com/up-the-hill/resonate-contacts.git
cd resonate-contacts
npm install
npm run dev
```

### pnpm (recommended)

```sh
git clone https://github.com/up-the-hill/resonate-contacts.git
cd resonate-contacts
pnpm install
pnpm run dev
```
